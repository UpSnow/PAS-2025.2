# Forma de testar o código

    Para testar partes específicas do código, comente o que não será usado no momento.

    Use o atalho do teclado:

    Ctrl + : (ou Ctrl + /, dependendo do editor)

    Isso transforma a linha selecionada em comentário, facilitando os testes sem apagar o código.

    Dica: você pode selecionar várias linhas e usar o atalho para comentar ou descomentar tudo de uma vez.
